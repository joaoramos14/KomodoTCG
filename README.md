# Komodo TCG

Site institucional fictício de uma empresa de compra, venda e avaliação de cartas Pokémon, desenvolvido como avaliação prática da disciplina de Desenvolvimento Web.

## Objetivo

Desenvolver um site utilizando apenas HTML, CSS e JavaScript, aplicando os conceitos trabalhados em aula: estruturação de páginas, manipulação do DOM, eventos, validação de formulários e organização de código. Não foi utilizado nenhum framework ou biblioteca (React, Angular, Vue, Bootstrap, jQuery, etc.).

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Estrutura das páginas

O projeto é dividido em 5 páginas:

- **index.html** - Home, com apresentação da empresa
- **sobre.html** - Sobre, com história, missão, visão e valores
- **servicos.html** - Serviços oferecidos (avaliação de cartas, montagem de decks, compra e venda, organização de campeonatos)
- **catalogo.html** - Catálogo de cartas, com busca e filtros
- **contato.html** - Formulário de contato

Todas as páginas compartilham o mesmo arquivo de estilo (`style.css`) e o mesmo menu de navegação.

## Funcionalidades implementadas

- Navegação entre as 5 páginas
- Busca de cartas pelo nome (catálogo)
- Filtro de cartas por tipo e por raridade (catálogo)
- Validação dos campos do formulário de contato (e-mail, tipo de solicitação, assunto e mensagem)
- Exibição de mensagem de sucesso após o envio do formulário, simulando o envio para um banco de dados
